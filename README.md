Quickstart
----------

1. Clone project.

2. Add products to installs directory.

3. Run 'init.sh' file.

4. Run web examples at [http://localhost:8080/optaplanner-webexamples-6.0.0-redhat-5] (http://localhost:8080/optaplanner-webexamples-6.0.0-redhat-5)
